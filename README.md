# aoc-cal
Calendary w/ programming languages of [Advent of Code](https://adventofcode.com/).
Currently supports only the last edition (2025) which lasts 12 days and stores data locally.

# Requirements
- [NodeJS](https://nodejs.org/en), version 25.2.1

# Features
- Save your answers and the programming language you used each day.
- Randomize the programming language to use by clicking the "Randomize" button.
